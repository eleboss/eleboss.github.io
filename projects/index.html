Toward Autonomous Rotation-Aware Unmanned Aerial Grasping

S. Lin, J. Wang, W. Yang, G. Xia.

IEEE Robotics and Automation Letters(RA-L), In submit

In this project, a vision-based autonomous UAM with a 3DoF robotic arm for rotational grasping and with a displacement compensation system for center of gravity compensation are proposed. And, a novel detection approach called Rotation-SqueezeDet is proposed to enable rotation-aware grasping, this approach can gives the target position and rotation angle in near real-time on Jetson TX2.

Tech: ROS C++ Python Tensorflow Solidworks Latex

[Open-hardware], [Video], [arXiv]

Autonomous Tunnel Inspection Using a UAV (Thesis, In progress)

Tech: ROS C++/C Python

An Intelligent Unmanned Aircraft System for Wilderness Search and Rescue

H. Yu, S. Lin, J. Wang, K. Fu, W. Yang.

International Micro Air Vehicles Conference and Flight Competition(IMAV), 2017. (Oral)

We combined infrared and optical target detection to increase the detection speed and accuracy and used multiple sensors to make this system can autonomous avoiding obstructions and landing on mobile platform. For further increase the Average Precision of SSD, we build a field people dataset UAV-PP and use ResNet-101 as the base net.

Tech: ROS C++/C Python Latex
